第 1 步：安装真正的 Python
打开浏览器，去这个地址下载：

https://www.python.org/downloads/

点那个黄色的大按钮下载，安装时 一定要勾选底部的 "Add Python to PATH"：


安装界面：
[x] Install launcher for all users
[x] Add Python to PATH   ← 这个必须勾！
装完后，重新打开一个终端（CMD 或 PowerShell），验证一下：


python --version
应该显示 Python 3.11.x 或 Python 3.12.x。

第 2 步：安装项目依赖
在项目目录下打开终端，运行：


cd C:\Users\20545\Desktop\xiaotian_quant
pip install -r requirements.txt
如果 pip 不识别，用 python -m pip install -r requirements.txt。

第 3 步：试运行

python main.py --mode backtest
如果看到类似下面的输出，就说明成功了：


2026-05-26 10:30:00 | INFO     | xtquant.engine         | 小天量化 v2.0 已启动
...
2026-05-26 10:30:05 | INFO     | xtquant.main           | 回测绩效报告
   total_return_pct: 12.35%
   sharpe_ratio: 1.85

报错
你的 Python 3.15 版本太新了，numpy 还没有预编译好的包。试试这个：


pip install numpy --only-binary :all:
如果还报错，说明 Python 3.15 确实没有预编译的 numpy。建议装 Python 3.12（最稳定，所有包都支持）：

卸载当前 Python，去 https://www.python.org/downloads/release/python-3129/ 下载 3.12
安装完重新 pip install -r requirements.txt
如果不想重装 Python，还有一个办法：跳过 numpy 编译，直接装预编译包


pip install --only-binary :all: numpy aiohttp aiosqlite
pip install pydantic pyyaml fastapi uvicorn websockets
