### 2.1 路由导航统一（P1-1）

**问题**：`IndicatorCommunity.tsx` 使用 `window.location.hash` 进行路由跳转，与项目其他页面使用的 `react-router-dom` `useNavigate` Hook 不统一。

**修改文件**：`web/src/pages/IndicatorCommunity.tsx`

**具体修改**：

```tsx
// 添加导入
+ import { useNavigate } from 'react-router-dom'

// 组件内添加 hook
+ const navigate = useNavigate()

// 替换 3 处 window.location.hash
- onClick={() => { window.location.hash = `#/indicator-ide?id=${indicator.id}` }}
+ onClick={() => { navigate(`/indicator-ide?id=${indicator.id}`) }}

- onClick={() => { window.location.hash = '#/indicator-ide' }}
+ onClick={() => { navigate('/indicator-ide') }}

- onAction={() => { window.location.hash = '#/indicator-ide' }}
+ onAction={() => { navigate('/indicator-ide') }}
```

**验证**：
```bash
grep -r "window.location.hash" web/src/pages/*.tsx
# 结果：无匹配，全部替换完成
```

| `IndicatorCommunity.tsx` | 1 | `alert(err.message)` → `toast('error', ...)` |

### 2.4 已确认无需修复的 P1 问题

#### A. Portfolio 日历导航方向 Bug

**审计报告描述**：左箭头代码是 `Math.min(...i + 1)`（下一月），右箭头是 `Math.max(0, i - 1)`（上一月），方向相反。

**实际代码状态**（`web/src/pages/Portfolio.tsx:229-240`）：

```tsx
// 左箭头（< ChevronLeft）：正确，索引减 1
onClick={() => setMonthIdx((i) => Math.max(0, i - 1))}

// 右箭头（> ChevronRight）：正确，索引加 1
onClick={() => setMonthIdx((i) => Math.min((months?.length || 1) - 1, i + 1))}
```

**结论**：该 Bug 已被修复，当前代码逻辑正确。
