
# 币富量化-CRA (Coin Rich AI) 软件使用说明 & 名词解释

## 一、产品概述

Coin Rich Ai (币富量化, 简称CRA) 是一个加密货币量化机器人工具，由量化团队和交易团队公测内测锤炼而成。

- **支持交易所**: Binance(币安)、OKX(欧易)、Bitget
- **支持交易类型**: 现货、合约(USDT本位)
- **核心功能**: 自动化策略交易、API管理、跟单、收益统计

## 三、现货策略参数

### 3.1 策略类型
| 策略 | 说明 |
|------|------|
| 马丁趋势策略 | 倍投补仓原理+趋势指标，倍投参数为2、4、8、16、32、64倍等 |
| 华尔街策略 | 等比数量补仓原理+趋势指标，补仓参数为1、2、3、5、8、13、21、34、55倍 |
| 激进策略 | (未详细说明) |

### 3.2 现货参数详解

| 参数 | 说明 | 范围/默认值 |
|------|------|------------|
| **挂单价格** | 0=实时市价买入；输入固定价格(如65000)则价格低于该值才买入 | 0或自定义价格 |
| **首单额度** | 首次开仓买入的金额，后续补仓按此基准翻倍 | 10-10000U |
| **首单加倍** | 首次买入按设置倍数买入，牛市时加倍吃利润 | 1倍起 |
| **循环类型** | 单次=止盈后不再做单；循环=继续做单，受循环次数限制 | 单次/循环 |
| **循环次数** | 最多做单次数 | 如100次 |
| **补仓次数** | 规划订单数量，一般建议5-7单 | 自定义 |
| **补仓总金额** | 所有补仓订单的总金额 | 如1280U |
| **补仓条件** | 设置跌幅比例和回调比例 | 自定义 |
| **防瀑布** | 1分钟内突然暴跌超过设定比例，暂停补仓，瀑布过后自动恢复 | 默认2% |
| **止盈方式** | 全仓止盈/尾单止盈/首尾止盈 | 三选一 |
| **止盈设置** | 静态止盈=固定比例+回调；移动止盈=动态根据盈亏比例 | 二选一 |
| **止盈参数** | 具体止盈比例和回调值 | 分仓止盈默认1.5% |

### 3.3 补仓说明
- 开启补仓后，达到设置的跌幅比例和回调比例，系统自动补仓
- 监控到1分钟内有突然暴跌，超过防瀑布比例，系统暂停补仓
- 等待瀑布过后，自动监控补仓

### 3.4 止盈方式详解
| 方式 | 说明 |
|------|------|
| **全仓止盈** | 整个仓位达到止盈比例后全部卖出 |
| **尾单止盈** | 只卖出最后一单的仓位 |
| **首尾止盈** | 只卖出首单和尾单的订单，合并出仓 |

## 四、合约策略参数

### 4.1 策略类型
| 类型 | 说明 |
|------|------|
| 顺势(多) | 跟随趋势做多 |
| 顺势(空) | 跟随趋势做空 |
| 逆势稳健 | 逆势操作，稳健风格 |
| 逆势激进 | 逆势操作，激进风格 |

### 4.2 合约参数详解

| 参数 | 说明 | 备注 |
|------|------|------|
| **开仓指标** | MACD/EMA/关闭 | 3个指标单选 |
| **MACD指标** | 金叉死叉作为开仓信号 | 金叉开多/补多，死叉开空/补空 |
| **EMA指标** | EMA60为标准线，均线以上做空单；EMA10+EMA60，60日均线上方做多 | 时间周期可选5/15/30/60分钟 |
| **策略方向** | 多/空/双向 | 合约区分方向，可买涨买跌 |
| **首单加倍** | 首次买入按倍数买入 | 牛市时加倍吃利润 |
| **顺势而为** | 多空同时开仓时，若一个方向被套，另一方向盈利循环做单时再次开仓，按被套次数+1倍开仓 | 对冲亏损 |
| **循环类型** | 单次/循环 | 同现货 |
| **循环次数** | 最多做单次数 | 如5000次 |
| **首单额度** | 首次开仓买入金额(保证金) | 如5U |
| **杠杆倍数** | 放大倍数 | 1-100倍 |
| **补仓指标** | MACD/EMA | MACD=金叉死叉条件补仓；EMA=EMA4上下拐点补仓 |
| **补仓次数** | 补仓订单数量 | 如9次 |
| **防瀑布** | 同现货 | 默认2% |
| **止盈方式** | 全仓/尾单/首尾 | 同现货 |
| **止盈设置** | 静态/移动 | 同现货 |
| **反向止盈** | MACD金叉开的多单，无补仓且盈利时，出现反向死叉信号则清仓 | 适合大周期订单 |
| **反向止损** | 出现反向信号且亏损状态，清仓卖出 | |
| **止损开关** | 开启后浮亏超止损比例则止损 | |
| **止损类型** | 比例止损/金额止损/价格止损 | 三选一 |
| **止损比例** | 如40% | |

## 七、核心名词解释

### 7.1 基础概念
| 名词 | 解释 |
|------|------|
| **做单数量** | 新建任务时规划的订单数量，建议5-7单 |
| **首单仓位** | 买入第一个订单的金额，范围10-10000U |
| **补仓价差** | 从首单买入价算起，每下跌达到设置百分比自动买入下一单，范围0.5-50% |
| **补仓回调** | 币价下跌到低点后上涨达到设置的回调比例才买入，范围0.01-0.5% |
| **止盈比例** | 订单成交达到设置的盈利百分比，分仓止盈常用1.3%，默认1.5% |
| **盈利回调** | 币价上涨到高点后下跌达到设置的回调比例才卖出，范围0.01-0.5% |
| **保证金可用余额** | 交易所可用来划转或补仓的资金 |
| **当前合约持仓** | 保证金余额 × 杠杆倍数 |
| **合约余额** | 当前账户的价值 |
| **当前现货持仓** | 现货购买后的金额 |

### 7.2 策略相关
| 名词 | 解释 |
|------|------|
| **马丁趋势策略** | 倍投补仓+趋势指标，倍投参数2、4、8、16、32、64倍等，补仓后持仓浮亏百分比下降一半 |
| **华尔街策略** | 等比数量补仓+趋势指标，补仓参数1、2、3、5、8、13、21、34、55倍 |
| **顺势而为** | 双向策略中，逆势单被套时，顺势单按逆势补仓次数+1倍开仓，对冲亏损。最高5倍 |
| **开仓加倍** | 首单金额×2倍，补仓倍数仍按首单金额计算 |
| **策略方向** | 多单(买涨)、空单(买跌)、双向(均买) |
| **交易次数** | 单次循环=止盈后不再买入(补仓正常)；策略循环=卖出后持续买入直到循环次数用尽 |
| **开仓指标策略** | 金叉开多(MACD金叉后买入)、死叉开空(MACD死叉后买入)、EMA开仓(行情拐点处自动开仓) |
| **补仓倍数** | 首单金额 × 倍数 |
| **防瀑布** | 1分钟内单一币种涨跌超过设定值，自动暂停补仓，默认2% |
| **清仓卖出** | 需先暂停订单，注意交易所手动清除仓位会导致清仓失败 |
| **一键补仓** | 输入金额直接市价买入 |
| **关闭补仓** | 启动后不执行补仓，但正常止盈 |
| **监控K线** | 监控币安MACD、EMA指标，周期5/15/30/60分钟/1/4/8小时 |
| **自定义减仓** | 极端行情下手动止损部分仓位，倍投补仓时最后一仓占50%，倒数第二仓25%，倒数第三仓12.5%... |
| **移动止盈** | 分三档/四档，第一档默认1.5%，止盈回撤30%。开启后分仓止盈、首尾止盈失效 |
| **MACD趋势建仓** | MACD值由负变正=金叉(开多/补多)；由正变负=死叉(开空/补空) |
| **限制在线单量** | 控制趋势开仓后进场的交易对数量，包括多单和空单数量，两个交易所绑定时为总和 |
| **保证金比例** | 维持保证金/保证金余额，达到100%时强平 |
| **盈利保护** | 盈利后实时从交易账户划转至资金账户(币安需开启万向划转权限) |
| **反向止盈** | MACD大周期金叉开多，小周期死叉卖出；补仓后失效；浮亏时执行原有止盈 |
| **反向止损** | MACD判断错误时直接止损 |
| **振幅** | 连续K线连续涨跌产生的价差幅度。5分钟建议2%，15分钟4%，30分钟7%，1小时10% |
| **燃料预警邮件** | 点卡低于设定值时发送邮件提醒 |
| **EMA和MACD补仓** | 行情下跌到设定百分比后，还需满足MACD或EMA条件才补仓，大行情时非常抗跌 |
| **启动顺势单补仓次数** | 补仓次数达到指定次数以上，开启对应顺势单，自动开启顺势而为 |
| **斩仓和燃烧** | ①双向燃烧：逆势单补仓到第3仓自动开启顺势单，盈利消耗逆势单浮亏；②超级全局燃烧：逆势单补仓到第5次，启动跨币种燃烧，所有盈利币种消耗逆势单浮亏 |
| **首单挂单补仓挂单价格** | 买入价格，最新价格达到挂单价格后系统自动市价买入 |

## 八、关键计算公式

```
当前合约持仓 = 保证金余额 × 杠杆倍数
保证金比例 = 维持保证金 / 保证金余额
维持保证金 = 仓位名义价值 × 维持保证金率
保证金余额 = 钱包余额 + 未实现盈亏

开仓额度(顺势而为) = 首单金额 × (逆势单补仓次数 + 1)
  - 逆势补仓1次 → 顺势首单开2倍
  - 逆势补仓2次 → 顺势首单开3倍
  - 逆势补仓3次 → 顺势首单开4倍
  - 逆势补仓4-7次 → 顺势首单开5倍(最高)

移动止盈计算:
  第一档: 止盈比例 × (1 - 30%回撤)
  第二档: 止盈比例 × (1 - 20%回撤)
  第三档: 依此类推
```

## 九、重要注意事项

1. **API绑定**: 支持币安、欧易、Bitget，API有效期永久
2. **激活流程**: 注册 → 充值USDT → 购买激活码 → 激活机器人 → 开始量化
3. **币安用户**: 盈利保护功能需开启万向划转权限
4. **清仓卖出**: 必须先暂停订单，交易所手动清除仓位会导致清仓失败
5. **自定义减仓**: 谨慎使用，最后一仓占比50%，倒数第二仓25%，倒数第三仓12.5%...
6. **移动止盈**: 开启后分仓止盈、首尾止盈功能失效
7. **反向止盈**: 首单补仓后自动失效；浮亏时执行原有止盈方式
8. **顺势单**: 不占用在线单数
9. **斩仓燃烧**: 小资金用户可能因交易所卖出金额限制导致燃烧不成功
10. **EMA周期**: 时间越长准确性越高，但也容易错过行情
